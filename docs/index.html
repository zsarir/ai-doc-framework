Get Started in Minutes

1

Install Framework

pip install ai-doc-framework

Or clone from GitHub and install dependencies

2

Run Setup Wizard

python tools/setup-wizard.py

Configure for single or multi-application project

3

Generate Structure

python tools/app-generator.py --type multi --apps web,api,admin

Create optimized documentation structure

4

Start Development

"Read START_TASK.md and execute task: [your task]"

AI automatically navigates and optimizes your workflow
