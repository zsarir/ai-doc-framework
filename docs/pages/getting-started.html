Quick Start Guide

1

Clone the Repository

Get the latest version of the framework from GitHub

Terminal 
git clone https://github.com/zsarir/ai-doc-framework.git
cd ai-doc-framework
2

Run the Setup Wizard

Use our interactive setup wizard to configure your project

Terminal 
python tools/setup-wizard.py
3

Generate Your Project Structure

Choose your project type and let the framework create the optimal structure

Terminal 
# For single application
python tools/app-generator.py --type single --name my-app

# For multi-application
python tools/app-generator.py --type multi --apps web,api,admin
