Installation Guide

Get started with the AI Documentation Framework in minutes. Choose your preferred installation method and follow our step-by-step guide.

Prerequisites

Ensure your system meets these requirements before installing the framework.

Python Environment

  • Python 3.8 or higher
  • pip package manager
  • Virtual environment support
  • Git for version control

System Requirements

  • Linux, macOS, or Windows
  • 50MB free disk space
  • Internet connection for installation
  • Terminal/Command prompt access

Development Tools

  • Git installed and configured
  • Text editor or IDE
  • Command line interface
  • Web browser for documentation

Installation Methods

Git Clone Method (Recommended)

This method provides the latest development version and allows easy updates.

1

Clone Repository

Download the framework from GitHub using Git clone.

Terminal 
git clone https://github.com/zsarir/ai-doc-framework.git
cd ai-doc-framework
2

Install Dependencies

Install the required Python packages using pip.

Terminal 
pip install -r requirements.txt
3

Run Setup Wizard

Execute the interactive setup wizard to configure your project.

Terminal 
python tools/setup-wizard.py
4

Verify Installation

Test that everything is working correctly.

Terminal 
python tools/validator.py --check-system

Direct Download Method

Download and extract the framework as a ZIP archive.

1

Download Archive

Download the latest release from GitHub releases.

Terminal 
wget https://github.com/zsarir/ai-doc-framework/archive/refs/tags/v1.0.0.zip
unzip v1.0.0.zip
cd ai-doc-framework-1.0.0
2

Install Requirements

Install the Python dependencies.

Terminal 
pip install -r requirements.txt

Pip Install Method

Install directly from PyPI (when available).

1

Install Package

Install the framework using pip.

Terminal 
pip install ai-doc-framework
2

Initialize Project

Initialize the framework in your project.

Terminal 
ai-doc-framework init

Docker Method

Run the framework in a Docker container.

1

Pull Docker Image

Pull the official Docker image.

Terminal 
docker pull ai-doc-framework:latest
2

Run Container

Run the framework in a container.

Terminal 
docker run -it --rm -v $(pwd):/workspace ai-doc-framework

Troubleshooting

Python Version Error

Problem: "Python 3.8+ required" error
Solution: Upgrade Python using your package manager or download from python.org

Permission Denied

Problem: Cannot install packages
Solution: Use pip install --user or run with sudo (not recommended)

Git Not Found

Problem: Git command not found
Solution: Install Git or use direct download method

Setup Wizard Fails

Problem: Wizard exits with error
Solution: Check Python path and permissions, try manual setup
