Document new errors using the standard template format

Error Template
# Error documentation format --- error_id: "[CATEGORY]-[SEVERITY]-[DATE]-[SEQ]" category: "[backend|frontend|infrastructure|security]" severity: "[critical|high|medium|low]" status: "[active|resolved|recurring]" --- ## Error Description [Describe the error in detail] ## Root Cause [What caused this error] ## Solution [Step-by-step resolution] ## Prevention [How to prevent this error in the future]

üîß Troubleshooting Guide

Python Version Error

Problem: "Python version too old" error
Solution:
# Check Python version python --version # Should be 3.8+ # Upgrade Python if needed # On Ubuntu/Debian: sudo apt update sudo apt install python3.9 # On macOS: brew install python@3.9

Permission Denied

Problem: Permission denied when creating directories
Solution:
# Fix permissions chmod +x tools/*.py chmod 755 . # Or run with sudo (not recommended) sudo python tools/setup-wizard.py

Missing Dependencies

Problem: "Module not found" error
Solution:
# Install missing dependencies pip install -r requirements.txt # For specific modules: pip install jinja2 pyyaml click colorama # Upgrade pip if needed: pip install --upgrade pip

AI Not Following Rules

Problem: AI not following AI_RULES.md guidelines
Solution:
# Check if AI_RULES.md exists ls -la AI_RULES.md # Validate rules file format python tools/validator.py --check-config # Ensure AI reads the rules first "Read AI_RULES.md carefully before starting any task"

Documentation Not Found

Problem: AI cannot find required documentation
Solution:
# Validate documentation structure python tools/validator.py --check-docs # Check documentation paths find docs/ -name "*.md" | head -10 # Regenerate missing documentation python tools/generate-docs.py

Performance Issues

Problem: Framework running slowly or consuming excessive resources
Solution:
# Enable performance monitoring export AI_DOC_FRAMEWORK_DEBUG=1 # Generate performance report python tools/generate-reports.py --format html # Run maintenance to optimize python tools/maintenance.py --weekly # Check system resources python tools/validator.py --check-system
